LAMP server installation

This repository contains scripts and required HTML files, Ubuntu/Debian installion scripts . The front app has a Responsive Html form that posts query by php POST method to backend .
The backapp has index.php file and reqquired LAMP server installation script . The servers were set up on two virtual machines,deployed on virtualbox ,  connected over as Internal Network 
and Host only network . The web servers were deployed on Apache2 server .
